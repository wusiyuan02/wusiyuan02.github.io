markdown语法介绍

markdown 是一种轻量级标记语言，文件后缀.md 就是markdown的缩写，使用易读易写的纯文本格式别写文档，可与HTML混编，markdown羽凡因其简洁、高效、易读、易写被大量使用，如github、wikipedia、简书等。

下面就记录学习markdown的基本语法。

Markdown 标题语法

在单词或短语面前添加（#），#的数量代表标题的级别。如添加三个#表示创建一个三级标题<h3>

Markdown 段落语法

用空白行将文本分隔，等同于

标签。 __注意:__不要使用空格和tabs缩进段落

Markdown 换行语法

在一行的末位至少添加两个或多个空格，然后按回车键，即可创建一个换行。<br />
也可以添加反斜杠（\）的方式实现换行，但有兼容性问题。

Markdown 强调语法

在要强调的部分前后加上两个（）或者两个（_）即可加粗文本
在要强调的部分前后加上一个（）或者一个（_）即可斜体显示文本
在要强调的部分前后加上三个（*）或者三个（_）即可同时加粗和斜体突出显示文本
建议使用*的形式来强调Markdown应用程序在处理单词和短语中间的下划线上并不一致，为考虑兼容性，在单词或短语中间部分强调的化请使用幸好。

Markdown 引用语法

在段落前添加>符号，即可创建块引用多个段落的块引用时，在段落之间的空白行也添加一个>符号。
嵌套块引用要在嵌套的段落前添加>>符号
块引用可以包含其他 Markdown 格式的元素。并非所有元素都可以使用。

Markdown 列表语法

有序列表的创建在每个列表项前添加数组并紧跟一个英文句点。数字不必按数学顺序排序，但列表应当从数字1起始。等价于HTML中的ol标签
无序列表请在每个列表项前面添加破折号 (-)、星号 (*) 或加号 (+) 。缩进一个或多个列表项可创建嵌套列表。等价于HTML中的ul标签。
列表中嵌套其他元素
段落引用块要在保留列表连续性的同时在列表中添加另一种元素，请将该元素缩进四个空格或一个制表符
代码块 代码块通常采用四个空格或一个制表符缩进。当它们被放在列表中时，请将它们缩进八个空格或两个制表符。

Markdown 分割线语法

在单独一行上使用三个或多个星号 (***)、破折号 (—) 或下划线 (___) ，并且不能包含其他内容。
为了兼容性在分割线的前后添加空白行。

Markdown 链接语法

超链接Markdown语法代码:[超链接显示名](超链接地址 “超链接title”)
链接文本放在中括号内，链接地址放在后面的括号中，链接title可选。
