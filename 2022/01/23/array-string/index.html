ArrayAndString

本人在实际原生js的编写的过程中，容易忘记相应的数组，字符串的方法使用。因此写下这篇笔记汇总数组和字符串常用的方法。

数组

数组是一种特殊的变量，它能够一次存放一个以上的值。对数组相同处理的方法也有很多种，比如将数组转换成字符串就有toString()/join()两种方法,但是这两种方法都可以把数组转化成字符串，但其中还是存在细节的差异。下面主要来介绍数组的方法并着重比较其中细节的差异。

数组的类栈、队列的方法

因为javascript的语法中没有栈和队列的数据类型，在JavaScript的语法中，就给数组创造了pop()、push()、shift()、unshift()的四种方法来模拟栈和队列的特性。

数组的增删改查

除去上一个标题记录的类栈、类队列的方法之外，下面记录的方法是对数组的增、删、改、查。故我不在该模块将上面记录的方法重复记录一遍。在这块，很多数组的方法都能产生相同的效果，但存在着细微的差异。

拼接方法splice()

splice()方法可用于向数组中添加新项、删除元素、拼接数组等等是一个功能十分强大的方法。
splice()方法第一个参数定义了应添加新元素的位置，第二个参数定义应删除多数元素，其他的参数定义了要添加的新元素。此方法返回一个包含已删除项的数组。

合并方法concat()

concat()方法通过合并现有数组来船舰一个新的数组，此方法不会更改现有数组，它总是返回一个新的数组。

concat()方法也可以将值作为参数用于数组的合并。

裁剪方法slice()

slice()方法用数组的某个片段切除新的数组。他和splice方法不同，他创建一个新的数组，不会从原数组中删除任何元素。
slice()可接受两个参数，第一个参数表示开始的位置，第二个参数表示结束的位置。(不包括结束的位置的元素)如果结束参数被省略，那会切除数组的剩余部分。

数组的遍历方法

这一类相比较前面的方法，在功能的使用和理解上面是有一定难度的。包括forEach()数组迭代方法、map()方法、filter()过滤器方法、reduce()方法、reduceRight()方法、every()方法、some()方法、indexOf()方法、lastIndexOf()方法、find()方法、findIndex()方法。
注意：除了 Internet Explorer 8 或更早的版本，所有浏览器都支持上述方法。

forEach()、map()、filter()

将这三个方法划分在一起，是因为这三种方法都为每个数组元素调用一次函数，都接受三个参数，三个参数的顺序都是(value、index、arr)，都返回一个数组。只不过，map和filter方法不会更改原始数组。
forEach()方法为每个数组元素调用一次回调函数，如果数组为空，就不会执行回调函数。
map()方法通过对每个数组元素执行函数来创建新数组，不会对没有值的数组元素执行操作，在这两方面来看，和forEach方法相同，不同的是，map()方法不会更改原始数组，他会返回一个新数组。
filter()方法也会对每个数组元素执行函数，但会返回通过测试的数组元素的新数组，这个方法在实际应用中更多的是用来筛选数组中的元素。

注意：这三个函数只接受三个参数，顺序为value,index,arr,如果仅使用value，可以忽略其他两个参数。

reduce()和reduceRight()

这两个方法使用方式相同，不同的是，reduce方法是从左向右，reduceRight方法是从右向左工作。此方法接受四个参数，总数（初始值/先前返回的值），value,index,arr。如果total有初始化的值，将总值写在回调函数外面。如果写在回调函数里面，默认是0。

every()和some()

这两种方法可以理解成为检测方法。返回的是boolean数据类型。当数组的元素都满足回调函数时，every返回true，否则返回false，当存在数组中的元素满足回调函数时，some返回true，否则返回false。

indexOf()、lastIndexOf()、find()、findIndex()
  • indexOf() 方法在数组中搜索元素值并返回其位置。第一个参数为预查找的元素，第二个参数可选，从哪里搜索。如果没找到返回-1.
  • Array.lastIndexOf() 与 Array.indexOf() 类似，但是从数组结尾开始搜索。返回的位置也是从结尾向前开始计数的。
  • find() 方法返回通过测试函数的第一个数组元素的值。回调函数的写法与filter类似，不同的是find只返回通过测试的第一个数组元素。
  • findIndex() 与 Array.find() 类似，方法返回通过测试函数的第一个数组元素的索引。

字符串

字符串和数组不同，字符串是原始值，数组是引用值。这说明字符串是无法使用方法和属性的。但是通过JavaScript，原始值也可以拥有方法和属性，这是因为在执行方法和属性的时候，JavaScript将其视为对象。
当想访问字符串上某一位置的字符时，也可以使用str[index]这种类数组访问的方法。

字符串的增删改查

字符串的查找

字符串的查找方法有indexOf()、lastIndexOf()、search()、match()、includes()、startsWith()、endsWith()共七种方法。

search()方法

indexOf()和lastIndexOf方法和数组一样，相比较于数组，字符串的search方法才是其一大亮点。
search()方法无法设置第二个开始位置的参数。但是，search可以使用正则表达式来查找特定的字符串并返回匹配位置。

match()方法

match() 方法根据正则表达式在字符串中搜索匹配项，并将匹配项作为 Array 对象返回。

includes()方法

includes()支持两个参数：searchvalue必需。需要搜索的字符串。start可选。默认为 0. 开始搜索的位置。返回： 如果字符串包含该值则返回 true，否则返回 false。

startswith() and endswith()

如果字符串以指定值开头，则 startsWith() 方法返回 true，否则返回 false。如果字符串以指定值结尾，则 endsWith() 方法返回 true，否则返回 false。这两个方法都区分大小写。
两者不同的是：startsWith()第二个参数是开始搜索的位置。endsWith第二个参数是要搜索的长度。

字符串的提取方法
提取字符串的方法

字符串有三种提取部分字符串的方法：**slice(start, end)、substring(start, end)、substr(start, length)**。substring()方法不同于slice方法，他无法支持负的索引。substr()方法与其他两种方法不同的是，他第二个参数规定被提取部分的长度。

提取字符串字符的方法

提起字符串字符可以使用str[index]这种形式来提取(属性访问)，不过这种方法是不安全的方法，更建议使用charAt和charCodeAt这两种安全的提取字符串字符方法。其中参数都是position。charAt返回字符串中指定下标的字符。charCodeAt 返回字符串中指定索引的字符的unicode编码。
属性访问：属性访问就是[]访问的方法，但使用属性访问有点缺点：

  1. 不适用 Internet Explorer 7 或更早的版本
  2. 它让字符串看起来像是数组（其实并不是）
  3. 如果找不到字符，[ ] 返回 undefined，而 charAt() 返回空字符串。
  4. 它是只读的。str[0] = “A” 不会产生错误（但也不会工作！）
字符串的替换

字符串替换使用replace方法，默认的replace()只替换首个匹配的字符(可以加g来全局匹配)，对大小写敏感(加i可以对大小写不敏感)。第一个参数是预替换的元素，第二个参数是替换后的元素。
replace()方法返回的是新字符串。第一个参数支持正则表达式的形式。
