Javascript重读（11）

JavaScript与HTML的交互是通过事件实现的，时间代表文档或浏览器窗口中某个有意义的时刻，可以使用尽在事件发生时执行的监听器（也叫处理程序）订阅事件。这以模型叫“观察者模式”。

事件绑定在HTML的元素上，事件应具有发生的顺序。事件流就是描述页面接收事件的顺序。

事件流

IE和Netscape提出了两种完全不同的事件流方案，IE支持事件冒泡，Netscape支持事件捕获。

事件冒泡

由目标元素向上找到父元素依次触发。

事件捕获

由根元素向下找他的子元素依次触发。

所有浏览器是从window对象开始捕获时间，但DOM2 event规范规定的是从document开始。

DOM事件流

事件冒泡和事件捕获的结合。

当达到目标元素的时候，状态由事件捕获转变成“到达目标“。

事件对象

DOM事件对象

事件对象是事件处理程序中的唯一参数。DOM0和DOM2的事件处理程序都有这个参数。

event中的属性有：

属性名（类型） 读/写 说明
eventPhase(number) 只读 表示调用事件处理程序的阶段：1 => 捕获，2 => 到达目标，3 => 冒泡
bubbles(boolean) 只读 事件是否可以冒泡
stopPropagation(function) 只读 取消所有后续事件捕获或者事件冒泡，当bubbles为true才可以调用
cancelable(boolean) 只读 是否可以取消事件的默认行为
preventDefault(function) 只读 取消事件的默认行为。当cancelable为true才可以调用
defaultPrevented(boolean) 只读 表示是否已调用preventDefault方法。true表示已调用
currentTarget(node) 只读 当前事件处理程序所在的DOM元素
target(node) 只读 事件目标
type 只读 触发的事件类型

在事件委托中，会用到currentTarget和target，其中currentTarget表示绑定事件的DOM元素。target表是真正触发时间的目标元素。

preventDefault()方法用于阻止特定事件的默认动作，举个例子就是a链接点击之后会跳转到href指定的URL，event.preventDefalut(）就可以取消这种默认动作。

IE事件对象

IE中的event对象是window对象的一个属性。

  • 对应DOM对象中的stopPropagation属性，IE事件对象中有cancelBubble。（可写属性）

  • 对应DOM对象中的preventDefault属性，IE事件中有returnValue。（可写属性）

  • 对应DOM对象中的target属性，IE事件中有srcElement。（可写属性）

事件类型

DOM3中定义了如下事件类型：

  • 用户界面事件：涉及与BOM交互的通用浏览器事件。
  • 焦点事件：元素获取或失去焦点是触发。
  • 鼠标事件：使用鼠标在页面上执行某些操作时触发。
  • 滚轮事件：使用鼠标滚轮时触发。
  • 输入事件：向文档中输入文本时触发。
  • 键盘事件：使用键盘在页面上执行某些操作时触发。
  • 合成事件：在使用某种IME输入字符时触发。

blur当元素失去焦点时触发，focus当元素获取焦点时触发。

鼠标事件

事件 说明
click 单击事件
dblclick 双击事件
mousedown 按下光标触发事件
mouseup 释放光标触发事件
mouseover 光标从元素外部移动到元素内部触发事件
mouseleave 光标从元素内部移动到元素外部触发事件
mouseenter 光标移动到元素内部触发事件
mousemove 光标在元素上移动反复触发
mouseout 光标从一个元素移动到另一个元素上触发事件

滚轮事件

在事件对象上有client-，page-，screen-共六个属性，分别表示光标距离客户端视口区域，页面区域，整个屏幕的坐标。

属性 说明
clientX 光标距离客户端视口的X坐标
clientY 光标距离客户端视口的Y坐标
pageX 光标距离整个页面视口的X坐标
pageY 光标距离整个页面视口的Y坐标
screenX 光标距离整个屏幕视口的X光标
screenY 光标距离整个屏幕视口的Y光标

输入事件

textInput，在其字符被输入到可编辑区域时触发。

键盘事件

事件 说明
keydown 用户按下键盘上某个键触发，而且持续按住会重复触发

可以通过event.keyCode来获取按下的键对应的键码。
